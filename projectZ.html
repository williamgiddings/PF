That Zombie Survival Game is a project that I began work on in 2016 for my A-Level Computer Science project. We were given 8 months to plan, design, create, test and evaluate our software.

The game itself was a wave-based first person zombie shooter which featured:

  • Intelligent AI with player stealth mechanics to allow for short-term breaks in the action to regain health and supplies
  • Ammo management
  • Simple weapon ballistics
  • Various different weapons of ranging ability, each being viable and balanced
  • Ragdoll physics which react to bullets and external force
  • Wave progression and difficulty incrementation

Tools used:

  • Unity Engine
  • C# (re-written from UnityScript)
  • Audio recorded/eddited with Audacity
  • Blender
  • Github
  • Email
